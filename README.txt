===== HƯỚNG DẪN CHẠY CHƯƠNG TRÌNH PROJECT 01 - DELIVERY SYSTEM - GROUP 09 =====
Để hỗ trợ quá trình chạy, đồ án có hỗ trợ GUI (Graphic User Interface) để đơn giản hóa quá trình thao tác, gồm các bước sau:
- Bước 1: Chạy file main.py
- Bước 2: Ta chọn level, input file (tại đây ta cứ chọn số thứ tự file input tương ứng, chương trình sẽ tự nhận biết level được chọn để đọc lên cho phù hợp), và tích chọn thuật toán muốn chạy.
- Bước 3: Bấm Run để thực hiện visualize thuật toán trên GUI, kết quả path sẽ được trả về và được ghi vô file output tương ứng.
*** Lưu ý: Đối với level 1, để có thể viết hết 5 path vô file output, GUI sẽ chạy cả 5 thuật toán, bắt đầu từ thuật toán được chọn và các thuật toán còn lại chạy theo thứ tự:
DFS → BFS → UCS → GBFS → A*.
